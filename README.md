# 📋 Portfolio Admin: Automated Ingestion Dashboard

A React and Firebase dashboard for project ingestion and portfolio management. Provides an interface for updating project metadata and assets in Firestore.

- ⚙️ **Project Ingestion**: Logic for adding new projects with automated metadata generation.
- 📊 **Firestore Dashboard**: Real-time management of project cards, categories, and tags.
- 📦 **Cloud Storage Integration**: Organization of project screenshots and files.

### Components
- **Admin Panel**: React-based UI for managing global portfolio state.
- **Ingestion Worker**: Logic for sanitizing and formatting project links and metadata.

**Deployment Context:**
Built as a private utility for managing a large-scale project library across multiple portfolio sites.
